# ft_printf
Recoding of the standard library function printf. Follows 42 norm standards.

<br><b>Supports the following format specifiers.</b><br>
| %s 
| %S 
| %d 
| %D 
| %i 
| %o 
| %O 
| %u 
| %U 
| %x 
| %X 
| %c 
| %C 
| %p 
<br>
<br>
<b>Supports precision, and minimum width.</b>
<br>
<br>
<b>Supported Flags</b>
<ul>
<li>#</li>
<li>0</li>
<li>+</li>
<li>-</li>
</ul>
<b>Supported Length modifiers.</b>
<ul>
<li>hh</li>
<li>h</li>
<li>l</li>
<li>ll</li>
<li>j</li>
<li>z</li>
</ul>
